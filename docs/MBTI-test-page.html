Myers Briggs Type Indicator Test

The Test

Based on the theories of Carl Jung, and developed by Isabel Briggs Meyer, and her mother Katharine Cook Briggs. The Myers Briggs Type Indicator test, determines, which of the 16 personality types you are.

How to Take the Test

For each statement, select the option that best describes you.
